Política de privacidad de Motiq

Última actualización: 19 de junio de 2026

Esta política describe qué datos recoge la aplicación Motiq ("la app"), cómo se usan y qué derechos tiene el usuario sobre ellos. Motiq es una app gratuita para llevar el control del mantenimiento de motocicletas según su kilometraje.

1. Responsable del tratamiento

Iván (desarrollador independiente de Motiq).
Contacto: ivan1905@gmail.com

2. Qué datos se recogen

Motiq recoge únicamente los datos que el propio usuario introduce para usar la app:

  • Cuenta: correo electrónico y contraseña (la contraseña se gestiona de forma segura por el proveedor de autenticación y nunca se almacena en texto plano).
  • Datos de tus motos: marca, modelo, año y kilometraje.
  • Historial de mantenimiento: tareas configuradas, lecturas de kilometraje, mantenimientos realizados, notas y coste que el usuario decida anotar.
  • Fotos de factura: imágenes que el usuario decide adjuntar como evidencia de un mantenimiento realizado.

Motiq no recoge datos de localización, contactos, identificadores publicitarios, ni utiliza analítica de terceros o redes de publicidad.

3. Para qué se usan los datos

Los datos se usan exclusivamente para el funcionamiento de la app:

  • Identificar al usuario y mantener su sesión iniciada.
  • Calcular cuándo corresponde cada tarea de mantenimiento según el kilometraje.
  • Mostrar el historial de mantenimientos y las fotos de factura asociadas.
  • Enviar notificaciones locales en el propio dispositivo cuando una tarea está próxima a vencer o vencida (estas notificaciones se generan en el dispositivo, no se envían notificaciones push remotas).

No se usan los datos para publicidad, perfiles comerciales, ni se venden o ceden a terceros con fines de marketing.

4. Dónde se almacenan los datos

Los datos se almacenan en Supabase, un proveedor de infraestructura (base de datos, autenticación y almacenamiento de archivos) que actúa como encargado del tratamiento. Las fotos de factura se guardan en un almacenamiento privado al que solo puede acceder el usuario propietario de cada moto (control de acceso a nivel de fila, Row Level Security).

No se comparten los datos del usuario con ningún otro tercero distinto del proveedor de infraestructura necesario para el funcionamiento técnico de la app.

5. Conservación y eliminación de datos

Los datos se conservan mientras el usuario mantenga su cuenta activa. El usuario puede eliminar cualquier moto y su historial asociado (incluidas las fotos de factura) directamente desde la app, en cualquier momento, desde la opción "Eliminar moto".

Para solicitar la eliminación completa de la cuenta y de todos los datos asociados, el usuario puede escribir a ivan1905@gmail.com.

6. Seguridad

El acceso a los datos está protegido mediante autenticación y políticas de seguridad a nivel de base de datos (Row Level Security), de forma que cada usuario solo puede ver y modificar sus propios datos.

7. Menores de edad

Motiq no está dirigida a menores de edad y no recoge intencionadamente datos de menores.

8. Cambios en esta política

Esta política puede actualizarse para reflejar cambios en la app. Cualquier cambio relevante se indicará actualizando la fecha de "última actualización" en este documento.
